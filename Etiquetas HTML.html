Etiquetas HTML

<h1> - <h6>

Las etiquetas <h1> a <h6> se utilizan para definir encabezados HTML

<p>

La <p>etiqueta define un párrafo.

<hr>

La etiqueta <hr> define una ruptura temática en una página HTML

<a>

La etiqueta <a> define un hipervínculo, que se utiliza para enlazar de una página a otra

<b>

La etiqueta <b> especifica texto en negrita sin ninguna importancia adicional

<i>

La etiqueta <i> define una parte del texto en una voz o estilo alternativo. El contenido interior normalmente se muestra en cursiva

<br>

La etiqueta <br>
inserta un salto de línea

<div>

La etiqueta <div> define una división o una sección en un documento HTML.
Se utiliza como contenedor de elementos HTML, a los que luego se les aplica estilo CSS o se manipulan con JavaScript

<nav>

La etiqueta <nav> define un conjunto de enlaces de navegación

<footer>

La etiqueta <footer> define un pie de página para un documento o sección

<button>

La etiqueta <button>define un botón en el que se puede hacer clic

<form>

La etiqueta <form> se utiliza para crear un formulario HTML para la entrada del usuario

<input>

La etiqueta <input> especifica un campo de entrada donde el usuario puede introducir datos

<textarea>

La etiqueta <textarea> define un control de entrada de texto de varias líneas

<img>

La etiqueta <img> se utiliza para incrustar una imagen en una página HTML
